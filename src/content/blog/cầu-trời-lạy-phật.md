Ngồi điền email mẫu của người ta mà mình run dã man.

Nhấn nút gửi rồi thì hạnh phúc khủng khiếp.

Nhưng lúc kiểm tra lại thì thấy mình gửi file gốc chứ không phải file pdf.

Mình đã gửi thêm file pdf.
